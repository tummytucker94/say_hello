# Project Overview
## “Say Hello Software” – Planning Phase

### 1. Problem Definition
I want to create software that greets the user when they enter their name because I want experience building software using the SDLC process.  
The purpose of this software is to greet people based on receiving their name as input.

---

### 2. Feasibility Study

**Technical Feasibility:**  
I have an M2 MacBook and plan to use Python with VS Code or PyCharm.

**Operational Feasibility:**  
This project satisfies my learning goal of completing a full SDLC cycle.

**Time Feasibility:**  
I plan to complete it within 3 days by November 2, 2025.

---

### 3. Objectives and Goals
- **Goal:** Gain experience building software using the SDLC process.  
- **Objective:** Build a software program that greets the user based on name input using SDLC.

---

### 4. Scope Definition
**In Scope:**  
- Accepting name input  
- Printing greeting  
- Handling blank input  

**Out of Scope:**  
- GUI design  
- Saving user data  
- Complex validation
