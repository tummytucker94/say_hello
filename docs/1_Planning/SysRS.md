# System Requirements Specification (SyRS)
## Project: “Say Hello Software”

### 1. System Overview
A console-based Python program that takes user input and displays a personalized greeting.

### 2. System Functions
1. Accept keyboard input
2. Store input in a variable
3. Print a message including that variable

### 3. System Inputs
- User-provided string (name)

### 4. System Outputs
- Printed message: “Hello [Name]”

### 5. Performance Requirements
- Response time: under 1 second
- System load: negligible

### 6. Hardware Requirements
- Any device that can run Python 3.x

### 7. Software Requirements
- Python 3.x interpreter
- Command-line or IDE interface

### 8. Interface Requirements
- Console text interface
